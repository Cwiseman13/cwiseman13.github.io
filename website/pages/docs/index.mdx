# Sonrai Documentation

Welcome to the Sonrai documentation! The documentation is reference material
for all available features and options of Sonrai DIG platform.

In the Quick Links below, you will find the most commonly used documentation
and a link to our guides that walk you through common tasks. Note that the
guides are located on the HashiCorp Learn site.

- Read more about the [configuration options](/docs/agent/options) for Sonrai Collector install options.
- Get started using Consul with our step-by-step guides at [Sonrai Knowledge Base](http://localhost:3000/docs).
